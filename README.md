# ia256utilities
***ia256utilities*** - is a set of useful utilities.

## Modules
* filesystem - load_json, save_json, save.

## Installation
### pip and git
$ pip install git+git://github.com/IceArrow256Studio/ia256utilities.git

### Just pip
$ pip install ia256utilities
